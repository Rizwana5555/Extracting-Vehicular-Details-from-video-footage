# Extracting-Vehicular-Details-from-video-footage
Project Overview

This project is a real-time Vehicle Detection and License Plate Recognition System developed using YOLO, PaddleOCR, and Gradio.

The system detects vehicles from images and videos, identifies the number plate region, extracts the text from the plate using OCR, and displays the results through a user-friendly web interface.

The project is designed for intelligent traffic monitoring and automated vehicle identification applications.

Features
Real-time vehicle detection
License plate localization
OCR-based number plate text extraction
Image and video input support
Web-based GUI using Gradio
Deep learning-based detection using YOLOv12
High-speed processing and accurate recognition
